# 俯视角2D射击建造防守游戏

俯视角2D射击+建造防守游戏。玩家控制角色移动射击，建造防御建筑阻挡并消灭从地图四周刷新的敌人。玩家HP归零则游戏结束。

## 技术栈

- 前端：React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- 桌面端：Tauri v2

## 开发与构建

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 类型检查 + 生产构建
npm run build

# 代码检查
npm run lint
```

桌面端开发/打包需要安装 Rust 工具链，然后使用：

```bash
npx tauri dev
npx tauri build
```

## 核心机制

### 玩家系统

- WASD/方向键移动，八方向
- 鼠标瞄准，左键射击（无限子弹，200ms冷却）
- HP: 100，被敌人攻击扣血
- 数字键1/2/3或UI按钮切换建造类型
- 鼠标点击空地放置建筑（有冷却）

### 敌人系统

- 从地图四周边缘随机生成
- AI: 寻路走向最近的玩家或建筑，优先攻击阻挡路径的建筑
- 近战攻击，接触时造成伤害
- 波次制：随时间增加刷新频率和数量

#### 敌人类型

| 类型 | 描述 |
|------|------|
| 普通僵尸 | 标准属性，红色 |
| 瘦猴僵尸 | 第2波起刷新，速度快、血量低、掉落更多，紫色 |
| 肥胖僵尸 | 第4波起刷新，血量高（普通僵尸4倍）、速度慢（普通僵尸一半）、体积大（普通僵尸2倍）、伤害高，橙色 |

### 建筑系统

1. 墙体：高HP(500)，阻挡敌人移动，吸引仇恨
2. 机枪塔：低HP(100)，自动攻击射程内敌人，每次射击消耗2HP耐久
3. 建材生产器：HP 50，2x2尺寸，造价100，每2秒生产1个建材

### 机枪塔瞄准机制

- 机枪塔自动选择射程内最近的敌人作为目标
- 墙体不再遮挡机枪塔视线，可隔着墙体攻击
- 圆形射程判定 + 带提前量的角度计算

### 子弹机制

- 玩家子弹与机枪塔子弹均可穿透墙体，不会被墙体阻挡

## 渲染层级

1. 地图背景（网格）
2. 建筑（墙体、机枪塔）
3. 敌人
4. 玩家
5. 子弹/特效
6. UI覆盖层

## 游戏循环

1. 生成新敌人
2. 更新敌人AI（寻路、移动、攻击）
3. 更新机枪塔（找目标、射击）
4. 更新子弹（移动、碰撞检测）
5. 更新玩家输入（移动、射击、建造）
6. 清理死亡实体
7. 检查游戏结束
